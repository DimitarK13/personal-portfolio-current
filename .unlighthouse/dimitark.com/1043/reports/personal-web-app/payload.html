Personal Web App.

Personal Web Application built for personal use. Modern simplistic UI design and a nice little challenge. This project took 3 months to fully complete and it consists of 3 pages: Notes, Reminders & Money Management. It automatically gets the time, date and day and displays a greeting message accordingly. The location is automatically found and displays the temperature.

Technologies:

  • React - Frontend
  • NodeJS - Server
  • ExpressJS - Backend
  • MongoDB - Database
  • OpenWeatherMap - API

Fonts and Colors:

  • Poppins - Primary Font
  • #008705 - Primary Color
  • #1E1F21 - Dark Color
  • #F8F8F8 - Light Gray Color
